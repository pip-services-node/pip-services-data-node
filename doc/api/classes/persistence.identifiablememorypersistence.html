Class IdentifiableMemoryPersistence<T, K>

Stores identifiable items of type T in memory and provides methods for working with the data that is stored.

An IdentifiableMemoryPersistence's max page size can be configured by passing ConfigParams with a "options.max_page_size" parameter to this class's configure method.
